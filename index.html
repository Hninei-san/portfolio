FEATURED CASE STUDY

Energy Consumption Analysis for Drone Delivery

This project is designed to predict the energy requirements of autonomous drones during package delivery lifecycles. By integrating high-fidelity telemetry data with physical and statistical modeling, the system achieves an understanding of flight dynamics and battery performance.

Technical Overview

  • Automatically detect takeoff, cruise, and landing phases using altitude-time profiles and Gaussian filtering.
  • Compute induced power and other energy-related metrics based on flight dynamics and environmental conditions.
  • Train a linear regression model that correlates physical power requirements with measured battery usage.
  • Estimate energy consumption for test flights and evaluate accuracy using Average Relative Error (ARE).

Analytics & Results

  • Evaluated CO₂ reduction potential, demonstrating drones as a sustainable alternative for last-mile delivery.
  • Achieved high precision regression results, validated against real-world test flight data using ARE metrics.
  • Identified key drivers of battery discharge, providing insights into how payload affects energy usage.
  • Developed a pipeline for processing telemetry data, enabling better path planning for autonomous missions.

AGENTIC AI CASE STUDY

AI Debugging Assistant

Modern software debugging is often time-consuming. This assistant leverages a multi-agent orchestration framework using LangGraph to automate the detection, analysis, and fixing of bugs within Python applications, simulating a senior developer's reasoning process.

Technical Overview

  • Orchestrates specialized agents (Scanner, Explainer, Fixer) utilizing LangGraph for complex iterative reasoning.
  • Employs advanced LLMs to perform deep semantic analysis of codebases to identify subtle logical errors.
  • Generates and validates precise code fixes, ensuring high reliability through integrated verification.
  • Uses feedback loops to continuously improve fix suggestions until the detected bug is resolved.

Analytics & Results

  • Significantly reduced debugging time for complex Python applications by providing clear explanations and fixes.
  • Successfully processed multi-file repositories with interdependent modules, demonstrating the robustness of the approach.
  • High success rate in identifying and fixing edge cases that traditional static analysis tools often miss.

COMPUTER VISION CASE STUDY

Automated pH Estimation

An automated system for pH level estimation using computer vision and machine learning, designed to provide accurate and real-time analysis of chemical indicator strips.

Technical Overview

  • Developed a CV-based pipeline that automatically segments pH indicator strips and extracts precise color values from test regions.
  • Implemented a robust normalization algorithm that uses a reference color chart to account for varied lighting conditions.
  • Trained regression models to map extracted RGB/HSV color vectors to precise pH values, achieving high numerical accuracy.
  • Optimized the image processing workflow to enable real-time pH estimation on embedded devices and mobile platforms.

Analytics & Results

  • Achieved a mean absolute error (MAE) of less than 0.1 pH units, consistently outperforming human subjects in controlled testing.
  • Successfully validated the system's performance across varied lighting environments through advanced white balancing.
  • Demonstrated the potential for low-cost, automated chemical analysis in agricultural monitoring applications.
Automatic segmentation of indicator regions from laboratory image data.
Regression model performance across the full pH spectrum.

COMPUTER VISION CASE STUDY

Height Measurement from a Single Camera View

A computer vision system designed to accurately estimate the height of objects or individuals from a single calibrated camera view using geometric perspective and reference objects.

Technical Overview

  • Implemented a geometric reasoning engine that utilizes vanishing points and horizon line detection to project real-world heights.
  • Developed an automated calibration routine using objects of known height to establish pixel-to-metric conversion factors.
  • Integrated YOLO-based models to automatically detect and segment individuals or objects for measurement within the frame.
  • Developed noise-reduction filters to account for lens distortion and camera tilt, ensuring stability across viewing angles.

Analytics & Results

  • Achieved a deviation of less than 3% compared to physical measurements at distances up to 10 meters.
  • Optimized for low-latency performance, enabling real-time height estimation for surveillance applications.
  • Demonstrated a streamlined setup process requiring only a single reference object for calibration.
Visualizing the geometric perspective lines and horizon detection for height inference.
